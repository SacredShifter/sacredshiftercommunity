# Sacred Shifter Community

A metaphysical software ecosystem for consciousness transformation.

## Overview

Sacred Shifter is a platform designed to facilitate spiritual growth, consciousness expansion, and community connection. It provides a suite of tools for personal reflection, shared experiences, and metaphysical exploration.

## Features

- **Sacred Feed**: Stream of consciousness transformation content
- **Sacred Circles**: Connect with fellow seekers in transformative group experiences
- **Mirror Journal**: Reflect, record, and analyze your inner journey
- **Resonance Register**: Document meaningful synchronicities and spiritual experiences
- **Personal Codex**: Your private collection of wisdom, insights, and revelations
- **YouTube Library**: Curated video content for consciousness expansion
- **Sacred Shifter Guidebook**: Ancient wisdom for modern transformation
- **AI Assistant**: Spiritual guidance powered by artificial intelligence

## Technology Stack

- React with TypeScript
- Vite for fast development
- Supabase for backend services
- Tailwind CSS for styling
- Shadcn UI components
- GSAP for animations
- Framer Motion for transitions

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file based on `.env.example`
4. Start the development server: `npm run dev`
